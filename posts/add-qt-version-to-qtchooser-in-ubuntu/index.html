How to add custom qt version to qtchooser in Ubuntu

May 5, 2020 One-minute read
  • Create new folder if it doesn’t exist:
mkdir ~/.config/qtchooser
  • Change directory:
cd ~/.config/qtchooser
  • Create new file:
qt_version.conf
  • Add to the created file following lines:
/path/to/qt/bin
/path/to/qt/lib
  • Create symbolic link to the created file:
ln -s qt_version.conf default.conf

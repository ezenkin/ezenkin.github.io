Mysql plugin for Qt under Ubuntu

May 28, 2020 One-minute read

Change directory with plugins source code

cd /opt/Qt/5.12.8/Src/qtbase/src/plugins/sqldrivers

Run qmake with path to mysql headers

qmake -- MYSQL_PREFIX=/usr

Run make to build mysql plugin

sudo make sub-mysql

Move to the directory where plugin will be located

cd /opt/Qt/5.12.8/gcc_64/plugins/sqldrivers

Copy plugin to that directory

cp /opt/Qt/5.12.8/Src/qtbase/src/plugins/sqldrivers/plugins/sqldrivers/libqsqlmysql.so .

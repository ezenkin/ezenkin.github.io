Reset mysql root password

September 16, 2020 One-minute read

Stop mysql server:

sudo systemctl stop mysql

Create a /var/run/mysqld directory

sudo mkdir -p /var/run/mysqld
sudo chown mysql:mysql /var/run/mysqld

Start mysql server in the safe mode:

sudo mysqld_safe --skip-grant-tables &

Run mysql client as root without password:

mysql -u root

Execute command:

mysql> UPDATE mysql.user SET authentication_string = PASSWORD('new_password') WHERE User = 'root' AND Host = 'localhost';

Exit the mysql client:

mysql> exit

Terminate current mysqld process:

sudo pkill mysqld

Start mysqld in normal mode:

sudo service mysql start
